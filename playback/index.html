--- title: Wiedergabe-API parent: Home description: '

Die Wiedergabe-API wird verwendet, um Video- und Wiedergabelistendaten von Video Cloud abzurufen. Es wird normalerweise verwendet, um Videodaten an einen Player auf einer Webseite oder in einer mobilen App zu übertragen. Die Playback API ist eine API mit niedriger Latenz, die für die clientseitige Verwendung beim Abrufen von Video- oder Playlist-Daten von Webseiten oder mobilen Apps vorgesehen ist. Es handelt sich nicht um eine allgemeine Medienverwaltungs-API, die Sie bei der Integration von Video Cloud in Ihr CMS oder andere Systeme verwenden können – dafür sollten Sie die CMS-API.

Die Authentifizierung für die Playback-API erfordert einen Richtlinienschlüssel, den Sie über die Richtlinien-API oder von einem Spieler in deinem Account.
