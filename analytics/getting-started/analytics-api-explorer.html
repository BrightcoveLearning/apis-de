Überblick

Die Analytics-API ermöglicht Ihnen das Abrufen verschiedener Analyseberichte für Ihre Video Cloud-Videos. Mit dieser Schnellstartanleitung können Sie eine Reihe verschiedener Anfragen testen, um zu sehen, wie die Anfrage-URL und die Antwort aussehen.

  • Eine vollständige Erklärung der API finden Sie im Übersicht über die Analytics-API und der Analytics-API-Referenz
  • Wenn Sie Anfragen auf Ihrem/Ihren eigenen Konten testen möchten, probieren Sie unsere API-Testtools

Genehmigung

API-Aufrufe werden mit OAuth2-Zugriffstoken autorisiert. Das Token wird mit Anfragen in einem Autorisierungsheader übergeben:

    Authorization: Bearer {access_token}

Zugriffstoken sind kurzlebig (5 Minuten), daher müssen Sie jedes Mal, wenn Sie eine Anfrage stellen, einen erhalten

In der Schnellstartanleitung beziehen sich alle Anfragen auf ein Brightcove-Beispielkonto mit a Stellvertreter das über die erforderlichen Client-Anmeldeinformationen verfügt und ein Zugriffstoken abruft und dann die Anforderung stellt.
