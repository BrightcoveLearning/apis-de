Einnahme

Um Ihrem Video Cloud-Konto eine Audiodatei hinzuzufügen, verwenden Sie die Dynamische Aufnahme-API oder der Modul hochladen. Ihre Quelldatei kann im MP3-, M4A- oder WAV-Format vorliegen. Der Aufnahmeprozess erfordert zwei Anfragen:

  1. Eine Erstellungsanfrage an die CMS-API um das Element in Ihrem Video Cloud-Katalog zu erstellen
  2. Eine Aufnahmeanfrage an die Aufnahme-API, um die Datei aufzunehmen und zu transkodieren

Einzelheiten zu diesen Anfragen finden Sie in der Referenz zur Dynamic Ingest API.

Aufnahmeprofil

Dynamische Lieferung

Wenn Ihr Konto für Dynamic Delivery aktiviert ist, benötigen Sie kein benutzerdefiniertes Profil für reinen Audioinhalt.

Legacy-Aufnahme

Wenn Ihr Konto das ältere Aufnahmesystem verwendet, verwenden Sie ein Profil, das reine Audiowiedergaben erstellt. Du kannst den ... benutzen audio-only Profil, das im . enthalten ist Standardprofile , oder Sie können ein benutzerdefiniertes Profil im Studio-Kontoeinstellungen oder mit dem Ingest Profiles API.

Wiedergaben

Die in Video Cloud für Ihr Audioelement erstellten Wiedergaben liegen im MP4- und HLS-Format vor, nur mit einer Audiospur. Das Element kann in einem Brightcove-Player wie ein Video abgespielt werden, wie erklärt unter.

Notiz: Um die Abdeckung der Wiedergabe auf mehreren Plattformen/Browsern zu maximieren, empfehlen wir, Audiodateien nur mit Audiotracks in MP4- und HLS-Wiedergaben zu transkodieren - Sie können MP3 nicht als Ausgabeformat angeben.

Poster und Miniaturansicht

Optional können Sie Ihrer Audiodatei ein Poster und/oder ein Miniaturbild hinzufügen. Das Poster wird während der Wiedergabe im Brightcove-Player angezeigt, und die Miniaturansicht wird in jeder Wiedergabeliste angezeigt, zu der Sie das Element hinzufügen.

Um Bilder zu Ihrem Audioelement hinzuzufügen, können Sie entweder Bearbeiten Sie die Elementeinstellungen in Studio oder benutze die Dynamische Aufnahme-API.

Musikplayer

Ihre Audioelemente werden in jedem abgespielt Brightcove-Player. Während der Wiedergabe ist der Videobildschirm entweder leer oder zeigt das Posterbild an, wenn Sie eins hinzugefügt.

Wenn Sie lieber einen minimalen Audioplayer ohne den Videobildschirm verwenden möchten, können Sie zu diesem Zweck einen Player erstellen. Nach dir den Spieler erstellen , Konfigurieren Sie die Größe auf folgende Art:

  • Verwenden Sie eine feste Größe
  • Verwenden Sie ein benutzerdefiniertes Seitenverhältnis
  • Stellen Sie die Höhe auf 80px ein (die Breite kann beliebig sein)
