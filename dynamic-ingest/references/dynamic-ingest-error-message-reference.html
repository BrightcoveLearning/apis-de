Fehlermeldungen

Die folgenden Fehlermeldungen können von der Dynamic Ingest API zurückgegeben werden

Fehlermeldungen bei dynamischer Aufnahme
Code Nachricht Typ Beschreibung
400 PROFILE json

Profil nach Namen nicht gefunden
400 BAD_REQUEST json

Anfragetext kann nicht geparst werden.
400 NOT_SUBMITTED json

Job kann nicht gesendet werden. Bitte versuchen Sie es später erneut
400 NO_SUCH_VIDEO json

Das referenzierte Video kann nicht gefunden werden
400 NO_SOURCE json

Keine Quelle zur Verwendung gefunden
400 CDN_CREDENTIALS json

CDN-Anmeldeinformationen können nicht abgerufen werden
400 BAD_CALLBACKS json

Rückrufe waren nicht im erwarteten Format
401 UNAUTHORIZED json

Authentifizierung fehlgeschlagen; Überprüfen Sie, ob Ihre Client-Anmeldeinformationen für das Zugriffstoken korrekt waren
403 DYNAMIC_DELIVERY_NOT_ALLOWED json

Dieses Konto ist nicht für Dynamic Delivery aktiviert, aber es wurde ein Dynamic Delivery-Profil angegeben
403 CONTEXT_AWARE_ENCODING_NOT_ALLOWED json

Dieses Konto ist nicht für kontextsensitives Encoding aktiviert, aber es wurde ein kontextsensitives Encoding-Profil angegeben
404 RESOURCE_NOT_FOUND json

Die API konnte die angeforderte Ressource nicht finden
415 UNSUPPORTED_MEDIA_TYPE json

Entweder ist die Datei, die Sie aufnehmen möchten, kein unterstützter Typ (siehe Unterstützte Video- und Audioformate, Codecs und Container ) oder die Anfrage enthielt kein a Content-Type: application/json Header
422 MALFORMED_SOURCE_URL json

Quell-URL ist fehlerhaft.
422 BAD_PROTOCOL_SOURCE_URL json

Quell-URL verwendet ein nicht unterstütztes Protokoll.
422 EXCEED_MAXIMUM_VTT_SOURCES json

vtt-Quellen überschreiten die maximale Größe.
422 INVALID_VTT_KIND json

vtt-Art ist ungültig.
422 CONSTRAINT_VIOLATION json

capture-image ist nicht erlaubt, wenn eine Bildquelle angegeben wird.
422 UNPROCESSABLE_ENTITY json

Anforderungsdaten enthalten eine nicht verarbeitbare Entität.
429 RATE_LIMIT_EXCEEDED json

Dynamischer Ingest-Job nicht erstellt. Verringern Sie die Anzahl gleichzeitiger Aufträge für dieses Konto, bevor Sie es erneut versuchen. Dieser Fehler wurde protokolliert
400 CDN_CONFIGS json

CDN-Anmeldeinformationen können nicht abgerufen werden
400 AMBIGUOUS_REQUEST json

In der Anfrage wurden sowohl eine Master-URL als auch use_archived_master festgelegt.
500 INGEST_ERROR json

Dieser Fehler weist im Allgemeinen darauf hin, dass der Aufnahmedienst vorübergehend unterbrochen wurde.
