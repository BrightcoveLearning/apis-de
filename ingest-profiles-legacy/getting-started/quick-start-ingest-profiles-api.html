Einführung

Die Ingest Profiles API ermöglicht Ihnen:

  • Holen Sie sich die Profile für Ihr Konto

  • Neue Profile erstellen

  • Profile aktualisieren

  • Profile löschen

  • Legen Sie das Standardprofil für Ihr Konto fest oder aktualisieren Sie es

Eine erholsame API

Die Ingest-Profile-API ist eine RESTful-API. Es akzeptiert HTTPS-Anfragen und gibt eine JSON-Antwort zurück.

Dies bedeutet, dass Sie die API mit jeder beliebigen Sprache verwenden können – Ruby, Java, PHP, Python, JavaScript (über AJAX oder nodejs) usw. Alles, was Sie tun müssen, ist HTTPS-Anfragen zu stellen und eine JSON-Antwort zu verarbeiten.

Wenn ein Benutzer ein Profil programmgesteuert abruft, um dieses Profil mit den abgerufenen Inhalten zu aktualisieren, muss der Benutzer das Profil bereinigen und Dinge wie "id" für Wiedergaben entfernen, aber nicht das Profil, und "version" sollte vollständig entfernt werden .

Die Basis-URL

Die Basis-URL für die Ingest Profiles API lautet:

    https://ingestion.api.brightcove.com/v1

Genehmigung

API-Aufrufe werden mit OAuth2-Zugriffstoken autorisiert. Das Token wird mit Anfragen in einem Autorisierungsheader übergeben:

    Authorization: Bearer {access_token}

Ein einzelnes Profil erhalten

Profil per ID erhalten

Um ein Profil nach ID abzurufen, stellen Sie eine GET-Anfrage an den Endpunkt:

    /accounts/{account_id}/profiles/{profile_id}

Für die profile_id , können Sie entweder Folgendes verwenden:

  • generierte ID (zB 54de14cce4b0a6d2bf9cb08a)

    https://ingestion.api.brightcove.com/v1/accounts/57838016001/profiles/screencast

Aktualisieren des Standardprofils

Aktualisieren Sie das Standardprofil

Um das Standardprofil für das Konto zu aktualisieren, stellen Sie eine PUT-Anfrage an den Endpunkt:

    /accounts/account_id/configuration

Fügen Sie diese JSON als Anfragetext ein

    {
      "id": {configuration_id},
      "account_id": {account_id},
      "default_profile_id": {default_profile_id}
    }

Die Konfigurations-ID wird zurückgegeben, wenn Sie das Standardprofil abrufen. Wenn kein Standardprofil vorhanden ist, siehe Einstellen des Standardprofils. Wie bei anderen Operationen können Sie den Namen oder die ID für die profile_id.

  • Name (zB hochauflösend)
  • generierte ID (zB 54de14cce4b0a6d2bf9cb08a)

Beispiel:

    https://ingestion.api.brightcove.com/v1/accounts/57838016001/configuration
