JobStartStop

Mit dem Endpunkt JobStartStop können Sie die Aktivierung und Deaktivierung eines vorhandenen Live-Jobs planen. Zum angegebenen Zeitpunkt führt unser System die erforderlichen API-Aufrufe durch, um einen Job in Ihrem Namen zu aktivieren und zu deaktivieren.

Anforderungen

  • Der Live-API-Schlüssel für Ihr Konto.
  • In Ihrem Live-Konto muss die Planerüberschreibung aktiviert sein.
  • Ein bestehender Live Job mit "statisch": wahr.

Erstellen Sie einen JobStartStop-Workflow

Nehmen wir an, wir möchten eine 2-stündige Veranstaltung planen, die um beginnt 2021-02-02T14:00:00+00:00.

Zeitstempel müssen im UTC-Format als Unix-Zeitstempel in Sekunden formatiert sein. Sie können verwenden https://www.unixtimestamp.com/ zur Konvertierung.

Beachten Sie, dass die beschriebenen Aktivierungszeitbeschränkungen Hier gelten weiterhin für geplante Aktivierungen, daher empfehlen wir, eine Zeit einzuplanen, die mindestens 60 Sekunden vor der gewünschten Startzeit liegt.

Für dieses Beispiel wählen wir 5 Minuten vor der beabsichtigten Startzeit.

Anfang: 2021-02-02T13:55:00+00:00 = 1612274100 Haltestelle: 2021-02-02T16:00:00+00:00 = 1612281600

Um den Workflow zu erstellen, stellen Sie die folgende Anfrage

POST https://api.bcovlive.io/v1/scheduler/jobstartstop Header: Content-Type: application/json X-API-KEY: Ihr API Key Body: { "job_id": "your Job ID", "activation": { "time_utc": 1612274100 } , "Deaktivierung": { "time_utc": 1612281600 } , "Bezeichnung": "Meine Dienstagssendung" }

Sie sollten eine ähnliche Antwort erhalten wie

{ "account_id": "your Account ID", "activation": { "state": "pending", "time_utc": 1612274100 }, "Deaktivierung": { "state": "pending", "time_utc": 1612281600 } , "Bezeichnung": "My Tuesday Broadcast", "job_id": "Ihre Job-ID", "type": "jobstartstop", "workflow_id": "d44cf29f4f184757ac0995beb8a0097e" }

Aktualisieren eines JobStartStop-Workflows

Sie können die Aktivierungs- und Deaktivierungszeiten eines JobStartStop-Workflows aktualisieren. Diese können nur aktualisiert werden, wenn "zustand": "ausstehend".

Nach dem vorherigen Beispiel möchten wir das Ereignis 30 Minuten früher als geplant beenden.

PUT https://api.bcovlive.io/v1/scheduler/jobstartstop/workflow_id Header: Content-Type: application/json X-API-KEY: Ihr API Key Body: { "deactivation": { "time_utc": 1612279800 } }

In der Antwort erhalten Sie den aktualisierten Workflow.

{ "account_id": "your Account ID", "activation": { "state": "pending", "time_utc": 1612274100 }, "Deaktivierung": { "state": "pending", "time_utc": 1612279800 } , "Bezeichnung": "My Tuesday Broadcast", "job_id": "Ihre Job-ID", "type": "jobstartstop", "workflow_id": "d44cf29f4f184757ac0995beb8a0097e" }

Abbrechen eines JobStartStop-Workflows

Sie können die Aktivierung und Deaktivierung mit a abbrechen LÖSCHEN Anfrage. Sie können dies nach der Aktivierung tun, um die Deaktivierung einfach abzubrechen.

LÖSCHEN https://api.bcovlive.io/v1/scheduler/jobstartstop/workflow_id Header: X-API-KEY: Ihr API-Schlüssel

In der Antwort erhalten Sie den aktualisierten Workflow.

{ "account_id": "your Account ID", "activation": { "state": "cancelled", "time_utc": 1612274100 }, "Deaktivierung": { "state": "cancelled", "time_utc": 1612279800 } , "Bezeichnung": "My Tuesday Broadcast", "job_id": "Ihre Job-ID", "type": "jobstartstop", "workflow_id": "d44cf29f4f184757ac0995beb8a0097e" }

Benachrichtigungen

Beim Erstellen des Workflows können Sie optional Benachrichtigungen konfigurieren. Sie müssen eine URL für unseren Service angeben POST zu. Wir senden eine Benachrichtigung, wenn der Scheduler die Aktivierungs- und Deaktivierungsaufrufe durchführt. Sie können auch so konfigurieren, dass Sie eine Benachrichtigung erhalten n Sekunden vor der Aktivierung und/oder Deaktivierung. Siehe die Benachrichtigungsleitfaden für mehr Details.

POST https://api.bcovlive.io/v1/scheduler/jobstartstop Header: Content-Type: application/json X-API-KEY: Ihr API Key Body: { "job_id": "your Job ID", "activation": { "time_utc": 1612274100, "notification": 600 } , "Deaktivierung": { "time_utc": 1612281600, "notification": 600 } , "Bezeichnung": "Meine Dienstagssendung", "notification_url": "http://example.com/live/jobstartstop/callbacks" }
