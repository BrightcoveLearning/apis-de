Einführung

Mit dem Live-Planer können Sie Aktionen planen für Static Entry Point (SEP)-Jobs.. Sie richten einfach einen Scheduler-Workflow mit den Zeiten ein und die notwendigen API-Aktionen werden automatisch abgewickelt. Es gibt zwei Arten von Scheduler-Workflows, die Sie einrichten können, wie in den folgenden Abschnitten beschrieben.

Planer starten/stoppen

Mit dem Scheduler können Sie Aktivierungs- und Deaktivierungszeiten für einen SEP-Job einrichten und die Zeiten auch aktualisieren. Siehe Details in Aktivierung/Deaktivierung eines SEP-Streams planen.

Clip-Planer

Sie können auch die Generierung eines Clips aus Ihrem Live-Job planen und den Zeitplan später aktualisieren oder den Clip abbrechen, wenn sich etwas ändert. Siehe Details in Planen der Clip-Generierung für einen Live-Stream.
