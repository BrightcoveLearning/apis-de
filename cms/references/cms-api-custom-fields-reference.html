Benutzerdefinierte Felder

Gebiet Typ Beschreibung
max_custom_fields Nummer Anzahl der benutzerdefinierten Felder, die ein Publisher im Konto erstellen kann
custom_fields Array Liste der benutzerdefinierten Feldobjekte im Konto
standard_fields Array Liste der Felder, die erforderlich sind oder eine Beschreibung haben. Standardfelder, die keines von beiden haben, werden in diesem Array nicht angezeigt.

Benutzerdefinierte Feldzuordnung

Für jedes benutzerdefinierte Feld:

Gebiet Typ Beschreibung
id Zeichenfolge Interner Name des Feldes
description Zeichenfolge  
display_name Zeichenfolge  
type Zeichenfolge einer von 'enum' oder 'string'' (was übersetzt "Liste" bedeutet' oder 'Text' in der Benutzeroberfläche)
enum_values Array Array von String-Werten, die für das benutzerdefinierte Feld gültig sind; nur vorhanden wenn type = 'enum'
required Boolesches Ob dieses Feld einen Wert haben muss, bevor das Video aktiviert werden kann
