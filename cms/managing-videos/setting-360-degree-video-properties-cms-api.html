Überblick

Um ein Video als 360°-Video zu identifizieren, muss die Projektionseigenschaft eingestellt werden. Sie können diese Eigenschaft entweder festlegen Verwenden des Medienmoduls in Studio , oder Sie können dies programmgesteuert tun, wie in diesem Thema gezeigt.

In diesem Beispiel wird die Postman-App verwendet, um Videoeigenschaften mit der CMS-API zu aktualisieren. Erfahren Sie, wie Sie mit dem beginnen Verwenden Sie Postman für HTTP-Anfragen dokumentieren.

In diesem Thema führen Sie die folgenden Aufgaben aus:

  • Kundenanmeldeinformationen abrufen
  • Videoinformationen abrufen
  • 360°-Videoeigenschaften festlegen

Kundenanmeldeinformationen abrufen

Um mit den Brightcove-APIs zu arbeiten, benötigen Sie Client-Anmeldeinformationen für das Konto und die API(s), die Sie verwenden möchten.

  1. Rufen Sie Ihre Kundenanmeldeinformationen in Studio ab, indem Sie den Anweisungen in folgen Anmeldeinformationen für die API-Authentifizierung verwalten. In den folgenden Schritten werden wir Spielerverwaltungs-API-Anfragen mit Postman stellen, daher sollten Ihre Anmeldeinformationen mindestens die folgenden Berechtigungen haben:

    • CMS: Video lesen/schreiben

    Sie können beliebig viele zusätzliche Berechtigungen hinzufügen, um Anmeldeinformationen zu erhalten, die für eine breitere Palette von API-Anfragen verwendet werden können. Beachten Sie auch, dass Sie Anmeldeinformationen erhalten, die für mehrere Konten funktionieren, wenn Sie möchten.

    Um Kundenanmeldeinformationen zu generieren, müssen Sie über Administratorberechtigungen für Studio verfügen oder eine Person in Ihrer Organisation mit Administratorberechtigungen beauftragen, die Anmeldeinformationen für Sie zu generieren.

Videoinformationen abrufen

Nachdem Sie Ihr Video in Video Cloud Studio hochgeladen haben, können Sie Videoinformationen von der CMS-API abrufen. Informationen zum Hochladen von Videos finden Sie im Hochladen von Videos mit dem Upload-Modul dokumentieren.

  1. Öffnen Sie die Postman-App.
  2. Rufen Sie ein neues Zugriffstoken für Ihr Video Cloud-Konto ab und wählen Sie das Token verwenden Taste.
  3. Mach ein WERDEN Anfrage an die CMS-API unter Verwendung der Video nach ID abrufen Syntax. Sie tun dies, indem Sie die folgende URL in das Anfrage-URL eingeben Feld, ersetzen Sie Ihre Konto-ID für {account_id} und deine Video-ID für {video_id}: 
        https://cms.api.brightcove.com/v1/accounts/{account_id}/videos/{video_id}
  4. Belassen Sie die Methode als GET und wählen Sie Senden:
  5. Im Antworttext sollten Sie das Projektion Eigentum.

360°-Videoeigenschaften festlegen

Um anzugeben, dass es sich bei einem Video um ein 360°-Video handelt, Projektion Eigenschaft sollte auf gesetzt werden gleicheckig.

  1. Kehren Sie zur Postman-App zurück. Möglicherweise benötigen Sie ein neues Zugriffstoken.
  2. Auswählen PATCH als HTTP-Methode, wobei dieselbe URL-Syntax wie im vorherigen Abschnitt verwendet wird.
  3. Drücke den Überschriften und geben Sie die Schlüsselwert Paar Inhaltstyp - Anwendung/json. Beginnen Sie einfach mit der Eingabe in den Eingabebereichen und Sie werden mit Optionen versorgt, aus denen Sie die angezeigten Werte auswählen können.
  4. Drücke den Körper Abschnitt direkt unter der URL, dann die roh Radio knopf:
  5. Geben Sie für die Rohdaten den folgenden JSON-Code für den Body ein (der Screenshot nach dem JSON zeigt, wie die Anfrage aussehen soll): 
        {
        "projection" : "equirectangular"
    }
  6. Klicken Senden
  7. Im Antworttext sollten Sie nun sehen, dass der Wert für die Projektion Eigenschaft eingestellt wurde.

    Jetzt können Sie Ihr Video als 360°-Video abspielen.
