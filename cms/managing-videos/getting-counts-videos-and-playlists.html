Einführung

Manchmal ist es hilfreich, die Anzahl der Videos oder Playlists in Ihrem Konto zu ermitteln oder zu erfahren, wie viele Videos im letzten Monat oder Jahr hinzugefügt wurden. Sie können diese Art von Zählungen ganz einfach mit dem /counts/ Endpunkt für die CMS-API.

Authentifizierung

Anfragen an die CMS-API benötigen einen Autorisierungs-Header, der Ihre Zugriffstoken enthält. Ausführliche Informationen zum Abrufen von Client-Anmeldeinformationen und deren Verwendung zum Abrufen von Zugriffstoken finden Sie im Übersicht über Brightcove OAuth.

Videozähler abrufen

Um eine Anzahl von Videos zu erhalten, machen Sie ein GET Anfrage zu:

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/videos

Die Antwort wird wie folgt aussehen:

    {
      "count": 4999
    }
Beachten Sie, dass gelöschte Videos nicht in der Zählung enthalten sind, oder in irgendwelchen CMS-API Antwort.

Was den Video-Zähl-Endpunkt so leistungsstark macht, ist, dass Sie auch a anhängen Suchparameter zur Anfrage. Auf diese Weise können Sie zählen, welche Videos die Suche zurückgeben würde. Da der Suchparameter so flexibel ist, können Sie viele Dinge zählen, wie zum Beispiel:

  • die Anzahl der Videos mit einem bestimmten Tag oder einem bestimmten Wert für ein benutzerdefiniertes Feld
  • die Anzahl der Videos, die im letzten Jahr (oder Monat oder Woche usw.) hinzugefügt wurden
  • die Anzahl der Videos, die derzeit abspielbar sind
  • die Anzahl der Videos, die nächste Woche veröffentlicht werden sollen

Anzahl der Playlists abrufen

Sie können auch die Anzahl der Playlists in Ihrem Konto abrufen, indem Sie a GET Anfrage zu:

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/playlists

Die Antwort hat das gleiche Format wie die für die Videozählung.

Sie können den Suchparameter auch in Anfragen zum Zählen von Playlists verwenden. So können Sie beispielsweise durch Anhängen herausfinden, wie viele manuelle Playlists sich im Konto befinden q=type:EXPLICIT zur Anfrage.

Darüber hinaus können Sie die Anzahl der Videos in allen manuellen oder intelligenten Wiedergabelisten abrufen, indem Sie Ihre GET-Anfrage an folgende Adresse richten:

    https://cms.api.brightcove.com/v1/accounts/{account_id}/counts/playlists/playlist_id/videos
