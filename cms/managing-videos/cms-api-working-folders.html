Einführung

Ordner können erstellt werden, um Videos in Ihrer Videobibliothek zu organisieren. Sie können Videos nach Thema, Autor, Produkt, Region usw. organisieren. Videos in Ordnern zu haben erleichtert die Verwaltung in einem Konto mit Hunderten oder Tausenden von Videos.

Anmerkungen:
  • Alle in diesem Thema behandelten Aufgaben können auch in Studio statt über die CMS-API.
  • Die CMS-API erfordert Client-Anmeldeinformationen und Zugriffstoken für alle Anforderungen. Wenn Sie Hilfe beim Abrufen von Anmeldeinformationen benötigen, lesen Sie die Abschnitt "Anmeldeinformationen abrufen" unten.
  • In diesem Thema werden die am häufigsten verwendeten Ordnervorgänge für die CMS-API behandelt, aber es gibt noch einige weitere, die Sie in der CMS-API-Referenz.

Um die zu verwenden CMS-API Sie benötigen die richtigen Anmeldeinformationen.

Die Anmeldeinformationen erhalten Sie in den meisten Fällen am einfachsten über den Abschnitt Authentifizierung der Studio-Admin-API (erfordert Administratorberechtigungen für Ihr Konto). Ordner erstellen

Hier sind die Informationen, die Sie benötigen, um einen neuen Ordner zu erstellen.

Anfrage-URL

Mach ein POST Anfrage zu:

      https://cms.api.brightcove.com/v1/accounts/account_id/folders

Anfragetext

      {
        "name": folder_name
      }

Antwort

Die Antwort wird wie folgt aussehen:

      {
        "id": "5a183efb10ab344b53775441",
        "account_id": "57838016001",
        "created_at": "2017-11-24T15:47:07.867Z",
        "name": "places",
        "updated_at": "2017-11-24T15:47:07.867Z",
        "video_count": 0
      }

Der Ordner id und name (der Name, den Sie im Anfragetext festgelegt haben) sind wichtig, da Sie beides benötigen füge dem Ordner ein Video hinzu. Mach dir keine Sorgen, wenn du das vergisst id , da Sie diese Informationen jedoch jederzeit wieder erhalten können, indem Sie die Ordner bekommen für das Konto.

Video zum Ordner hinzufügen

Um ein Video zu einem Ordner hinzuzufügen, machen Sie ein PUT Anfrage zu:

      https://cms.api.brightcove.com/v1/accounts/account_id/folders/folder_id/videos/video_id

Anfragetext

Für diesen Vorgang gibt es keinen Anforderungstext.

Antwort

Die Antwort auf diese Anfrage lautet 204 NO CONTENT.

Video aus Ordner entfernen

Um ein Video aus einem Ordner zu entfernen, machen Sie ein DELETE Anfrage an dieselbe URL, die Sie zum Hinzufügen eines Videos verwenden:

      https://cms.api.brightcove.com/v1/accounts/account_id/folders/folder_id/videos/video_id

Anfragetext

Für diese Anfrage müssen Sie auch den Ordnernamen in den Anfragetext aufnehmen:

      {
        "name": folder_name
      }

Holen Sie sich Videos in einen Ordner

Um die Videos in einen Ordner zu laden, machen Sie ein GET Anfrage zu:

      https://cms.api.brightcove.com/v1/accounts/:account_id/folders/folder_id/videos

Parameter anfordern

Grenze und Versatz

Standardmäßig gibt diese Anforderung die 20 Videos zurück, die dem Ordner zuletzt hinzugefügt wurden. Sie können dies ändern und die Ergebnisse für einen Ordner mit vielen Videos durchblättern, indem Sie einen oder beide dieser Parameter zur Anfrage hinzufügen:

  • limit: die Anzahl der zurückzugebenden Videos (Standard: 20; Maximum: 100)
  • offset: die Anzahl der zu überspringenden Videos (Standard: 0)

Diese Anfrage gibt beispielsweise Videos 21-30 der möglichen Ergebnisse zurück (vorausgesetzt, der Ordner enthält so viele Videos):

https://cms.api.brightcove.com/v1/accounts/:account_id/folders/folder_id/videos?limit=10&offset=20

Sortieren

Sie können die zurückgegebenen Videos auch sortieren, indem Sie die Sortieren Parameter und setzen Sie ihn auf eines dieser Felder:

  • Name
  • Referenz ID
  • aktualisiert am
  • hergestellt in
  • veröffentlicht_at
  • schedule_starts_at
  • schedule_ends_at
  • Zustand
  • spielt_total
  • play_trailing_week

Um in absteigender Reihenfolge zu sortieren, stellen Sie dem Feldwert ein Minuszeichen (-hergestellt in).

Der Standardwert für Sortieren ist -aktualisiert am.
