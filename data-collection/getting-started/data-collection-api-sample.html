Hinweise zur Probe

Der Code für dieses Beispiel besteht ausschließlich aus JavaScript und besteht aus zwei Teilen (beide im CodePen: Das Plugin-Skript befindet sich oben und ein kurzer Code, der das Plugin zum Player hinzufügt unten):

  • Das Skript, das das Plugin zum videojs-Player hinzufügt
  • Das Skript für das Plugin, das Daten an den Data Collector sendet

Beachten Sie, dass die Daten an die Data Collection API "gesendet" werden, indem Sie die Anforderungs-URL als src Wert für ein neues Bild (img )-Element und injizieren dieses Element dann in die Seite.

Sie sollten die gleiche Methode verwenden, anstatt XMLHttpRequest oder etwas anderes um zu vermeiden CORS Themen.
