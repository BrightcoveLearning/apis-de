Einführung

Beim Erstellen von Liveereignissen mit dem Live-Modul müssen Sie ein Live-Aufnahmeprofil auswählen, das die Anzahl und Qualität der von Brightcove Live erstellten Wiedergabeversionen bestimmt. Dieses Thema enthält detaillierte Informationen zu jedem der Live-Standardaufnahmeprofile.

Anmerkungen:
  • Für Jobs, die über die Live-API erstellt wurden, können Sie derzeit kein Aufnahmeprofil angeben, sondern müssen stattdessen die Ausgabemerkmale definieren.
  • Diese Profile werden speziell beim Erstellen von Live-Events verwendet. Sie werden beim Hochladen von Videos in Video Cloud nicht verwendet.

Es ist auch möglich, eigene benutzerdefinierte Live-Aufnahmeprofile zu erstellen. Informationen finden Sie unter Erstellen von benutzerdefinierten Live-Ingest-Profilen.
