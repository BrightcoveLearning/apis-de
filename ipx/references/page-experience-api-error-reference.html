In-Page Experience API-Fehler

In-Page Experience API-Fehlerreferenz
Fehler Beispiele Beschreibung
400 BAD_REQUEST 
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "No template specified"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "message": "Specified template does not exist"
}
{
  "status": 400,
  "code": "BAD_REQUEST",
  "details": [
    "Path `name` is required."
  ]
}
 In der Regel ein Problem mit dem Anforderungstext, z. B. das Weglassen der template oder ein anderes erforderliches Feld oder die Angabe einer nicht vorhandenen Vorlage
401 UNAUTHORIZED 
[
  {
    "error_code": "UNAUTHORIZED",
    "message": "Permission denied."
  }
]
 Entweder wurde mit der Anforderung kein Zugriffstoken gesendet oder die zum Erstellen des Zugriffstokens verwendeten Client-Anmeldeinformationen hatten nicht die richtigen Berechtigungen für die Anforderung
404 NOT_FOUND 
{
  "status": 404,
  "code": "NOT_FOUND",
  "message": "Experience {experience_id} not found"
}
 Die Erfahrung, die Sie aktualisieren möchten, ist nicht vorhanden. Sie wurde möglicherweise gelöscht oder die Erfahrungs-ID in der Anfrage ist möglicherweise falsch.
