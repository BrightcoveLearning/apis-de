Einführung

Die Brightcove-Bild-API bietet eine schnelle und einfache Möglichkeit, ein Bild mithilfe einer einfachen GET-Anforderung in der Größe zu ändern, zuzuschneiden und/oder zu drehen.

Anfrage-URL

Die Anforderungs-URL für die Image-API lautet:

https://images.brightcovecdn.com/image/v1/account_id/default/url/{image_url}

Die erforderlichen Pfadparameter sind:

  • account_id: Ihre Video Cloud-Konto-ID
  • image_url: eine öffentliche URL, unter der das Bild zu finden ist
Anmerkungen:
  • Die folgenden Bildformate werden unterstützt: PNG, JPG, GIF
  • Die image_url muss richtig codiert sein. Wenn Sie JavaScript verwenden, verwenden Sie die encodeURIComponent() Methode. Sie können auch die String-Encoder bereitgestellt von Brightcove Learning Services:

URL-Parameter

Um eine transformierte Version Ihres Bildes zu erhalten, verwenden Sie einen oder mehrere der unten in Ihrer Anfrage beschriebenen URL-Parameter. Beachten Sie, dass Sie eine beliebige Kombination der 3 Parameter verwenden können.

URL-Parameter für Bild-API
Parameter Beschreibung
resize Ändern Sie die Größe des Bildes, indem Sie dies gleich "widthxheight" setzen - Beispiel: Größe ändern=160x90)
crop Beschneiden Sie Breite und Höhe, indem Sie dies gleich "widthxheight" setzen - Beispiel: Zuschneiden=160x90 - Der Ursprungspunkt ist die Mitte des Bildes.
rotate Bild um die angegebene Gradzahl im Uhrzeigersinn drehen - Beispiel: drehen=90

Empfohlene Vorgehensweise

Wenn Ihre transformierten Bilder auf Webseiten oder in Online-Apps angezeigt werden, verwenden Sie die API, um die Bilder abzurufen und auf Ihrem Server zu speichern. Tun nicht Rufen Sie sie zur Laufzeit ab, da eine Geschwindigkeitsbegrenzung der API dazu führen würde, dass einige Anfragen auf Seiten oder Apps mit hohem Datenverkehr fehlschlagen, und außerdem die Leistung Ihrer Seite/App beeinträchtigt, indem die API-Anforderungen bei jedem Laden von erzwungen werden Ein Benutzer.
